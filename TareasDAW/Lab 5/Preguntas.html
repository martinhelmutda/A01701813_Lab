Cuestionario

  • ¿Qué es entonces una base de datos - DB?

    • Es un método usado para el almacenamiento estructurado de datos. Son ficheros interrelacionados, con estructuras complejas y compartidas por varios procesos de forma simultánea (unos online y otros por lotes) [1]. Representa entidades y sus interrelaciones

  • ¿En qué casos es conveniente usar bases de datos?

    • Cuando se necesita hacer un sistema de información que procese información con múltiples tipos de datos interrelacionados. Evita la redundancia de datos y la fácil edición de éstos

  • ¿Qué es un sistema de gestión de base de datos - DBMS?

    • Es conocido también como (SGDB), consiste en una colección de datos interrelacionados y un conjunto de programas para accerder a los mismos. [2]

  • Enlista y explica las funciones/responsabilidades que tienen los DBMS.

    • Son encargadas de el almacenamiento, edición y consulta de datos

    • Gestionar transacciones de datos

    • Flexibilidad e independencia de datos

    • Reducen redundancia

    • Permitir el acceso a varios usuarios a la vez

  • Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

    • TopTec, un programa que realizamos para la materia de diseño operativo. Usar una base de datos nos hubiera beneficiado increiblemente la gestión de datos por cada campus. Teníamos todo tipo de datos para cada uno y era muy cansado y complicado meterlos directamente en HTML
