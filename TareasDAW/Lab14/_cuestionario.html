  • ¿Qué es ODBC y para qué es útil?
    • OPEN DATABASE CONNECTIVITY. Son especificaciones que permiten el acceso a las bases de datos desde cualquier software de manejo de bases de datos.
  • ¿Qué es SQL Injection?
    • Es una forma de ataque malicioso en donde el infiltrado ejecuta sentencias de SQL para controlar la base de datos del servidor.
  • ¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?
    • La revisión rutinaria de la base de datos, así como el uso de aplicaciones para encontrar vulnerabilidades Agregar credenciales necesarias ppara cada función. Verifica los inputs antes de hacer el query.
