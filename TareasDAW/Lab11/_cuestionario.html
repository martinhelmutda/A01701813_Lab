Cuestionario

  • ¿Por qué es una buena práctica separar el controlador de la vista?.
    • Permite manejar aplicaciones web de una forma modular, sencilla de entender y mantenible. Permite realizar cambios en un lugar sin que los otros sean afectados.
  • Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?
    • $_COOKIE. HTTP Cookies
    • $_REQUEST. HTTP Request variables
    • $_ENV. Enviromental Variables
    • $_FILES
    • $_SERVER. Server and execution environment information
    • $_SESSION. Session variables
  • Explora las funciones de php, y describe 2 que no hayas visto en otro lenguaje y que llamen tu atención.
    • glob() Es una funcion que me permite ubicar variables de otros documentos del proyecto
    • register_shutdown_function() Es una funciión que ejecuta una acción justo antes de que un script termine
  • ¿Qué es XSS y cómo se puede prevenir?
    • Es el escribir código dentro de inputs. Podemos prevenirlo usando specialcharts, strip_tags, así como PHP Input filter
